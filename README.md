# Gemini-Spotify-Roaster
Input your Spotify username and Google Gemini will roast all of your public playlists and songs on your profile.

Spotify API setup in "spotify_roast.py":
  - Create a Spotify App to use the API, for 'client_id', 'client_secret', and 'redirect_uri' variables

Gemini API setup in "spotify_roast.py":
  - Create a Gemini App to use the API, for 'api_key' variable
  - Can change roast prompt given to Gemini in 'prompt' variable
